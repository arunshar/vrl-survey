ARR May reviews in (Overall 3 / 2.5 / 2 / 1.5). Plan: post author responses by Jul 13 AoE, skip the Aug 2 EMNLP commitment, wait for the Jul 30 meta-review, then submit the expanded version to CSUR (arXiv preprint right after Jul 30). Full strategy in the decision memo below.

## Layout

- [docs/ARR_author_responses.md](docs/ARR_author_responses.md): paste-ready responses to all four reviewers (red-teamed). **Post before Jul 13 AoE.**
- [docs/coauthor_decision_memo.md](docs/coauthor_decision_memo.md): paste-ready memo for the co-author thread.

- [paper-acl/](paper-acl/): the submitted 8-page ACL ARR source (frozen reference).
- [paper-csur/](paper-csur/): the expansion (acmart, 14 sections; each stub carries its writing brief).
- [scripts/verify_bib.py](scripts/verify_bib.py): citation-integrity gate (arXiv id check, arXiv title search, Crossref fallback). `--selftest` for offline checks.
- [scripts/workflow_lit_ingestion.js](scripts/workflow_lit_ingestion.js): the Phase C ingestion workflow script.

## Rules

- No citation enters `paper-csur/references.bib` unverified (`scripts/verify_bib.py` must pass).
- No em dashes in prose; en dashes only for numeric ranges.
- Nothing is posted to OpenReview, arXiv, or ManuscriptCentral by automation; authors submit.
